# Extensions
Blockyfish Client comes with two Chrome Extensions that can be toggled on or off in the settings. It relies on [electron-chrome-extensions](https://github.com/samuelmaddock/electron-browser-shell) to provide the necessary Extension APIs. 

## uBlock Origin
Blockyfish Client uses uBlock Origin for blocking ads in the game. Source for uBlock Origin is [here](https://github.com/gorhill/uBlock). 

## Doctorpus Assets
Blockyfish Client comes with support for Doc-Assets, a cute asset pack made by [Doctorpus](https://github.com/The-Doctorpus). Its source can be found [here](https://github.com/The-Doctorpus/doc-assets)
